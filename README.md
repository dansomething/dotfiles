dotfiles

My .files for OS X, Linux, and Windows

1. Git >=1.7
2. [RubyGems][0] >=1.3.6
2. Vim >=7.3

## Installation
1. bash <(curl -s https://raw.github.com/dansomething/dotfiles/master/install)
